ballet.project module¶

class ballet.project.Project(package)[source]¶

Bases: object

Encapsulate information on a ballet project

This is a utility class mostly useful for easy access to the project’s information from within the ballet.validation package.

In addition to the defined methods and properties, the following functions of the project can be accessed as attributes of a class instance, where prj refers to the python module of the underlying ballet project: - load_data (prj.load_data.load_data) - build (prj.features.build) - collect_contrib_features (prj.features.collect_contrib_features)

property on_master_after_merge¶

Check the repo HEAD is on master after a merge commit

Checks for two qualities of the current project: 1. The project repo’s head is the master branch 2. The project repo’s head commit is a merge commit.

Note that fast-forward style merges will not cause the second condition to evaluate to true.
