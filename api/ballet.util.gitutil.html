ballet.util.gitutil.get_diffs_by_diff_str(repo, diff_str)[source]¶

Get file changes via a diff string.

For details on specifying revisions, see git help revisions.

Parameters:
  • repo (git.Repo) – Repo object initialized with project root
  • diff_str (str) – diff string identifying range of diff. For example, master..HEAD diffs from master to HEAD, and 12345678..abcdef90 compares to commits.
Returns:

list of git.diff.Diff identifying changes between revisions
